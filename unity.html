Inventory Display

Unity 3d UniInventory

https://github.com/fbxiang/InventoryDemo

RPG style inventory system

I found inventory system a good start point for any kind of game. The existing inventory systems are lacking extendability. Therefore I designed my own inventory system to be programmer friendly:

  • Define Item as an abstract base class for great power via proper inheritance
  • Use the model-view-controller model for easy control over different inventories
  • Design data structures for almost unlimited data storage into any single item
  • provide customizable tool tip display
  • Animation and sound for interaction with items
  • Well handled game update for each item
  • Callback style item usage definition
