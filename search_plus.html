Search-plus Chrome Extension

https://github.com/fbxiang/SearchPlus

A Chrome extension for multi purpose search

This extension is designed to support more ways for searching information within a web page. Currently 3 search modes are supported:

  • normal search
  • multi-word search
  • regular expression search

Normal search is the basic case insensitive word matching search. Multi-word search let you search for multiple words at the same time. Regular expression search matches javascript regular expression with content in the page.

An in-browser script executor

This extension exposes the javascript environment to the user. Using the code mode. Any javascript code will be evaluated.

I always find it annoying when I need to run some short script quickly. I have to open up text editors and run commands from the terminal. Now, run some short commands while browsing? No problem! Though it has tremendous power, you can call it a calculator and do some simple math with it.

A customizable panel

By default, there are built in commands to navigate between different modes. "/multi" for multi-word search. "/code" for code execution.

You want more commands? No problem. "/addcmd" to add your own command. Command is based on javascript so you can do anything. Please refer to the project page(on Github) for details.
