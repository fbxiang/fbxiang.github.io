CUMTD Real-time bus schedule

This app supports most needed functions for bus search in the Champaign-Urbana area:

  • Nearby bus stop listing
  • Real-time bus schedule
  • Real-time bus location
  • Bus stop search
  • Favorite bus stops
  • Arrival alarm for buses
  • Bus trip planning

The main focus of this app is to gain experience of user interface design. So some of the functions are not totally implemented. We followed the iterative design process. We analyzed the previous app designs and recognized problems, did prototyping at various fidelity and user studies.

Future work

The user interface design is mostly completed as the requirement of this course. In the future, we will further improve the back end functionality to complete the app.
